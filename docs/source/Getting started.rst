Getting Started
===============

This document will show you how to get up and running the web interface. The whole process is very straightforward but it does require an initial plan about the network requirements from the user side.The process is staright forward and intuitive.

This section is divided into two parts: one for users who want to make use of the platform for their use and another for the developers who want to understand and carry on with the development of the platform.
